## 3D Video Advanced Previewer V1.0.0
SBS (Side-by-Side) や TAB (Top-and-Bottom) 形式の3D動画を、ブラウザ上でリアルタイムに補正して2D表示・プレビューするためのツールです。

## 主な機能
- **形式変換:** 左右分割(SBS)・上下分割(TAB)の動画から、一方の視点のみを抽出して表示。
- **独立補正システム:** 左右(上下)それぞれの映像に対して、個別に位置(X/Y)・ズームを微調整可能。
- **重ね合わせ(Overlay)モード:** 左右の映像を半透明で重ねて表示し、ズレや歪みを正確に分析可能。
- **同時調整(Link)機能:** リンクをONにすると、両側のパラメータを同期して調整可能。
- **長押しリピート:** 調整ボタンを押し続けることで、連続的に数値を変更可能。
- **アスペクト比補正:** 圧縮された映像(Half-SBS等)を正しい比率(16:9等)へ自動補正。
- **プライバシー配慮:** サーバーへ動画を送信せず、すべてブラウザ内のローカル環境で処理。

## 技術スタック
- **Frontend:** HTML5, JavaScript, CSS3
- **Styling:** Tailwind CSS
- **Drawing:** Canvas API

## 使いかた
1. `index.html` をブラウザで開きます。
2. 3D動画ファイルを画面中央にドラッグ＆ドロップします。
3. 動画の形式（SBS / TAB）を選択します。
4. 設定パネルから位置やズームを調整し、理想的な表示状態に整えます。
5. 重ね合わせモードを活用して、左右の映像の視差（くぼみ・飛び出し）を微調整してください。
